1. Data Collection

Data collection is the first step in machine learning. It involves gathering relevant data from different sources. The quality of data affects the performance of the model. In this project, student performance data was collected for analysis and prediction.

2. Data Cleaning

Data cleaning is the process of checking and correcting errors in the dataset. Missing values, duplicate records, and incorrect data are handled in this step. Clean data improves the accuracy of machine learning models. Our dataset was already clean and ready to use.

3. Feature Selection

Feature selection means choosing the input variables that help predict the target variable. In this project, Study Hours, Attendance, and Assignments were selected as features. These factors influence student marks. Selecting relevant features improves model performance.

4. Train-Test Split

The dataset is divided into training and testing data. The training data is used to teach the model, while the testing data is used to evaluate its performance. This helps measure how well the model works on unseen data.

5. Model Training

Model training is the process where the algorithm learns patterns from the training data. Linear Regression was used in this project. The model learned the relationship between study habits and student marks.

6. Testing

After training, the model is tested using the test dataset. Predictions are generated and compared with actual values. Testing helps determine whether the model has learned correctly.

7. Evaluation

Evaluation measures how well the model performs. Metrics such as Mean Absolute Error (MAE) and R² Score are used. Lower MAE and higher R² values indicate better performance.

8. Prediction

Prediction is the final step where the trained model is used to estimate outcomes for new data. In this project, the model predicted student marks based on study hours, attendance, and assignments.
